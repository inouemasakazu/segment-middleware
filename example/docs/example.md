# example
## 1. 概要
セグメントLED制御用ミドルウェアの実例について説明する。  
実例ではダイナミック制御方式で構成された4桁のセグメントLEDを、コマンドラインユーザーインターフェース(以降、CLI)を介した制御を実現するためのサンプルコードとHW構成等を記載する。  


### 1.1. 評価環境
使用するデバイス、HWを以下に記載する。  

| 名称 | 説明 | 備考 |
|---|---|:-:|
| マイコン | NUCLEO-F446RE | STM32F446RE搭載の評価ボード |
| セグメントLED | ダイナミック接続対応の4桁7セグメントLED | [秋月電子通称 -商品ページ](https://akizukidenshi.com/catalog/g/g103673/) |
| u74hc595 | シフトレジスタ | [秋月電子通称 -商品ページ](https://akizukidenshi.com/catalog/g/g114053/) |
| その他 | 抵抗・ブレッドボード等 | - |


開発・動作確認時の環境を以下に記載する。

| 名称 | 説明 | 備考 |
|:--|---|:-:|
| STM32CubeIDE | マイコン用の統合開発環境(IDE) | ver.2.1.0を使用 |
| Windows 11 | 開発マシンのOS | - |
| Tera Term | ターミナルソフト | CLI用に使用、 baudrate:115200 |

## 2. ソフトウェア構成
### 2.1 HALの設定
各ペリフェラルの設定を以下に示す。  
※設定はCube MXを使用  

| 名称 | 説明 | 備考 |
|:--|---|:-:|
| PA7 | 1桁目の点灯/消灯制御用端子に割り当て | - |
| PB8 | 2桁目の点灯/消灯制御用端子に割り当て | - |
| PB9 | 3桁目の点灯/消灯制御用端子に割り当て | - |
| PA6 | 4桁目の点灯/消灯制御用端子に割り当て | - |
| PC1 | シフトレジスタ制御用端子に割り当て | SER端子に接続 |
| PC2 | シフトレジスタ制御用端子に割り当て | SRCLK端子に接続 |
| PC3 | シフトレジスタ制御用端子に割り当て | RCLK端子に接続 |
| TIM6 | 周期割り込み用タイマーとして使用(1ms周期) | - |
| UART | Tera Termとの接続用IF | baudrate:115200 |


### 2.2. ファイル構成
サンプルコード用に定義・流用したファイルとその構成を以下に示す。  

```text
project/
  ├── example/
  |     ├── CLI/
  |     |    ├── cli.c
  |     |    └── cli.h
  |     |
  |     ├── u74hc595/
  |     |    ├── _u74hc595.c
  |     |    └── _u74hc595.h
  |     |
  |     ├── UART/
  |     |    ├── uart_config.h
  |     |    ├── uart_hal.h
  |     |    ├── uart.c
  |     |    └── uart.h
  |     |
  |     ├── cmd.c
  |     ├── cmd.h
  |     ├── display.c
  |     └── display.h
  └── segment
        ├── segment.c
        └── segment.h
```

* Note  
1. 「CLI/u74hc595/UART」配下は各機能用に定義された、ミドルウェア・ライブラリ用モジュールを使用する。  
2. 「cmd.c(h)」はコマンドラインユーザーインターフェース用アプリケーションとして定義する。  
3. 「display.c(h)」はセグメントLED制御用ミドルウェアを使用するアプリケーションとして定義する。  

## 3. 動作例
### 3.1. サンプルコード
サンプルコードはフォルダ「example」配下を参照すること。  


### 3.2. CLIコマンド一覧

```shell
# LEDに数値を設定
# 以下は、LEDに1234を表示する例
seg set 1234

# 以下は、LEDに30.0を表示する例
seg set 30.0

# 点滅ON 500[ms]で点滅する例
seg blink on 500

# 点滅OFF
seg blink off

# 表示クリア
seg clr

# "seg" → "segment"でも同様の動作が可能
segment set 1234
```
